The product

Two boards click together. No solder.

  • The brain — a 14-port Motherboard with a color TFT scoreboard, a speaker, and two click-in rotary knobs, each ringed with 12 LEDs. Runs the OS + every game.
  • The screen — twelve "Light Grid" NeoPixel tiles = a 6×18, 108-pixel RGB matrix. A dumb, glowing renderer.
  • On-device OS boots into 10 games + ambient modes. Play: Pong, Snake, egg-catch, Flappy, Invaders, a jukebox… Idle: a clock, a visualizer, a drifting pixel scene.
Platform, not a toy

A full console + 10 games, built in a day.

One engine, two homes

Every game is dependency-free C++ that compiles unchanged into the ESP32 firmware and a desktop simulator — so the whole library is testable off-hardware, deterministically.

Themeable to the core

5 runtime themes restyle the whole console — menu, every game, the knob rings, the glow itself — from design tokens. Never a hardcoded hex.

Clean seams

Ports-and-adapters: the brain owns logic and draws to an abstract Canvas; the screen just glows. Adding a game is one class.

Room to grow

Roadmap: an installable game "market" (browse + install) and code-your-own cartridges — the razor's blade.

The margin is real

The Duo's parts cost ~$292 at Atech retail (12 Light Grids alone = $108) — so no third party can build a $249 console from them. At Atech's internal COGS ~$120–175 → 30–52% gross margin. (COGS is an estimated band.)

…then it keeps earning

Hardware sells once; Arcade Pass + game packs earn forever at ~95% margin, on rails Atech already owns silicon-to-store.

Why Atech wins

Every unit pulls twelve Light Grids through the door.

  • Module pull-through. One Duo books 2 Motherboards + 12 Light Grids + 2 Knobs + a Screen + a Speaker + 2 USB-C — five module types, twelve of the one nobody buys twelve of.
